Aurelia Muse class-reservation 수정 안내

수정 파일:
- class-reservation.html
- class-reservation.css
- class-reservation.js

적용 방법:
1. 기존 프로젝트의 class-reservation.html, class-reservation.css, class-reservation.js를 백업합니다.
2. 이 압축 파일 안의 세 파일을 기존 프로젝트 폴더에 덮어씁니다.
3. 브라우저에서 class-reservation.html을 새로고침하여 확인합니다.

수정 내용:
- “Aurelia Muse 클래스 신청” 옆 일러스트를 더 풍성한 접시·찻잔·리본 무드로 개선했습니다.
- 01, 02, 03 숫자 크기를 크게 조정했습니다.
- 예약 정보 입력 카드가 PC에서 sticky로 따라오고, 마우스 이동 시 은은하게 반응하는 효과를 추가했습니다.
- Choose Your Atelier / 클래스 코스 선택 영역의 제목, 부제목, 설명 간격을 줄였습니다.
- 코스 선택 시 예약 금액 자동 계산 기능을 유지했습니다.
- 모바일에서는 sticky 효과를 해제해 자연스럽게 1단으로 표시되도록 했습니다.
